# PRD-121: US News Rankings

Status: Draft

## Data Asset Contract

- Source Authority: US News published rankings.
- Product Value: Supports U.S.-market discovery, comparison, and counselor-facing explanation.
- User-visible Fields: To be defined only after source review.
- Verification Requirement: Ranking claims must retain source, category, year, rank, and tie context.
- Freshness Requirement: To be defined by publication cycle.
- Versioning Rule: Annual rankings must remain year-specific.
- Known Limitations: Source methodology and access boundaries must be documented before use.
- Downstream Consumers: Ranking comparison, university discovery, reports, commercial packaging.
