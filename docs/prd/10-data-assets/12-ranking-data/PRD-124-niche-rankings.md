# PRD-124: Niche Rankings

Status: Draft

## Data Asset Contract

- Source Authority: Niche published rankings or ratings.
- Product Value: Supports student-facing perception, fit, and qualitative comparison when appropriate.
- User-visible Fields: To be defined only after source review.
- Verification Requirement: Ranking or rating claims must retain source, category, year or release context.
- Freshness Requirement: To be defined by source availability.
- Versioning Rule: Source-specific rating and ranking changes must remain traceable.
- Known Limitations: Methodology and user-generated signal limitations must be documented before decision use.
- Downstream Consumers: Discovery, fit explanation, reports.
